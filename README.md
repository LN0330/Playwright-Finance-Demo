# Playwright-Finance-Demo  
### 使用 Playwright 實現金融數據抓取，並產出csv檔案、截圖、trace file  
- Demo重點: Playwright爬蟲實作、測試檔案產出、XPath  
- Playwright-Finance-XPath.py版本: 透過辨識特定文字或XPath實現直接定位抓取版本  
---
- 撰寫語言: Python 3.13.1  
- 套件: Playwright  
- 開發環境: Visual Studio Code
